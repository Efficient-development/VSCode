### 快捷键操作

    Command(⌘) + W : 关闭当前文件
    Command(⌘) + W : 退出VSCode
    Command(⌘) + P : 文件跳转操作
    control(⌘) + ⬆️ + tab : 在所有打开的文件中跳转
    Command(⌘) + ⬆️ + p : 快速打开命令面板
    Command(⌘) + ⬆️ + o : 跳转到文件中的符号
    control(⌘) + G : 跳转到文件中的某一行
    Command(⌘) + T: 跳转到工作区中的符号

    在快速打开命令面板后，如果输入了? 那么就可以列出所有可用的相关命令。
